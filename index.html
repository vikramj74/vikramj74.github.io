Don't Pick The Last One

Dont Pick The Last One is a fun single player web based game. The player plays against the CPU. Each player on their turn are required to pick up some sticks from a pile. The player who picks the last stick loses the game. This game was developed using HTML, CSS and JavaScript libraries like JQuery and JQuery-UI. The code for the game can be found on my github profile.

LINK : DontPickTheLastOne

RegistrationPro

RegistrationPro, is a software solution, which could be used by any College / University Society, Interest Group or event managers, to quickly create a database of the students registered for any event, using any android device (smartphone, tablet etc.). The application has two distinct modules, register student and find a student. The register student module can be used to register a student, by making him fill up a form, and then clicking his picture. These details will now be stored on the mobile device. Now the other module, find a student can be used to retrieve the registration details and a picture of the student.

Airline Reservation System

The Airline Reservation System is an online airline route management system as well as a ticket reservation system. The system has two modules, administration and customer. The administration module can be used by the airline management staff, to edit, view and manage the route, flight and cost details, of each flight. The customer module can be used by any customer to create an online customer profile, view various flight routes / costs and book any no. of flight tickets. The complete system was based on the java applet technology, which made it possible to do all the above tasks online.
